Acasă Simple Structurate

Variabile Simple

  • Aceste variabile rețin un singur element de date și sunt cele mai elementare tipuri de variabile. Exemple de variabile simple includ:

  • Tipuri întregi (int, short, long, long long, unsigned int etc.)

    • int varInt = 10;

    unsigned int varUInt = 15;

  • Tipuri reale (float, double, long double)

    • float varFloat = 3.14f;

    double varDouble = 3.14159;

  • Tipuri caracter (char)

    • char varChar = 'A';

  • Tipuri booleene (bool)

    • bool varBool = true;
