Acasă Simple Structurate

Variabile Structurate

  • Acestea sunt variabile care pot conține mai multe elemente, fiecare element putând fi de tip diferit (în structuri, de exemplu). Tipuri de variabile structurate includ:
  • Vectori

    • În C++, vectorii sunt o variantă modernă și mai flexibilă a tablourilor. Vectorii fac parte din biblioteca standard (<vector>) și sunt preferați față de tablourile statice deoarece:

      Pot crește sau micșora dinamic.

      Oferă funcții utile pentru manipularea datelor (inserare, ștergere, sortare etc.).

    Exemplu de inițializare:

    #include <vector>
std::vector<int> vectorInt = {1, 2, 3, 4, 5};

    Adăugare element:

    vectorInt.push_back(6); // Adaugă 6 la sfârșitul vectorului

    Accesare element:

    int val = vectorInt[2]; // Accesează al treilea element (val = 3)
  • Matrice

    • O matrice este un tablou bidimensional (sau multidimensional), în care datele sunt organizate pe linii și coloane. Este o extensie a tabloului unidimensional (sau vector).

    Definiție:

    int matrice[3][4]; // Matrice cu 3 linii și 4 coloane

    Inițializare:

    int matrice[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

    În acest exemplu, matricea are 2 linii și 3 coloane, unde fiecare element poate fi accesat folosind doi indici, de exemplu: matrice[0][1] pentru accesarea valorii 2.
