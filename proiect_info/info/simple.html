Variabile Simple

  • Aceste variabile retin un singur element de date si sunt cele mai elementare tipuri de variabile. Exemple de variabile simple includ:

  • Tipuri intregi (int, short, long, long long, unsigned int etc.)

    • int varInt = 10;

    unsigned int varUInt = 15;

  • Tipuri reale (float, double, long double)

    • float varFloat = 3.14f;

    double varDouble = 3.14159;

  • Tipuri caracter (char)

    • char varChar = 'A';

  • Tipuri booleene (bool)

    • bool varBool = true;

  • Probleme propusa CMMDC

    • Cerinta:

    Sa se scrie un program care sa determine cel mai mare divizor comun a doua numere naturale citite de la tastatura.

    Date de intrare:

    Programul citeste de la tastatura doua numere naturale a si b.

    Date de iesire:

    Programul afiseaza pe ecran numarul x, cel mai mare divizor comun al numerelor a si b.

    Restrictii si precizari:

    0 <= a, b < 1.000.000.000

    daca ambele numere sunt egale cu 0 se va afisa valoarea -1

  • Problema propusa numar prim

    • Cerinta

    Sa se scrie un program care citeste de la tastatura un numar natural n si verifica daca este prim.

    Date de intrare

    Programul citeste de la tastatura numarul n.

    Date de iesire

    Programul afiseaza pe ecran mesajul DA, daca n este numar prim, respectiv NU in caz contrar.

    Restrictii si precizari

    0 <= n <= 1.000.000.000

    un numar natural este prim daca are exact doi divizori distincti: 1 si el insusi.

  • Problema propusa Fibonacci

    • Cerinta

    Se da un numar natural n. Sa se afiseze in ordine crescatoare, primii n termeni ai sirului lui Fibonacci.

    Date de intrare

    Programul citeste de la tastatura numarul n.

    Date de iesire

    Programul afiseaza pe ecran primii n termeni ai sirului lui Fibonacci, in ordine crescatoare, separati printr-un spatiu.

    Restrictii si precizari

    1 <= n <= 40

  • Problema propusa Cifre

    • Cerinta

    Calculati suma cifrelor numarului dat.

    Date de intrare

    Programul va citi de la tastatura un numar natural.

    Date de iesire

    Programul va afisa pe ecran suma cifrelor numarului citit.

    Restrictii si precizari

    numarul citit va fi mai mic decat 2.000.000.000
