Acasa Simple Structurate

Variabile Structurate

  • Acestea sunt variabile care pot contine mai multe elemente, fiecare element putand fi de tip diferit (in structuri, de exemplu). Tipuri de variabile structurate includ:
  • Teorie
  • Vectori

    • In C++, vectorii sunt o varianta moderna si mai flexibila a tablourilor. Vectorii fac parte din biblioteca standard (<vector>) si sunt preferati fata de tablourile statice deoarece:

      Pot creste sau micsora dinamic.

      Ofera functii utile pentru manipularea datelor (inserare, stergere, sortare etc.).

    Exemplu de initializare:

    #include <vector>
std::vector<int> vectorInt = {1, 2, 3, 4, 5};

    Adaugare element:

    vectorInt.push_back(6); // Adauga 6 la sfarsitul vectorului

    Accesare element:

    int val = vectorInt[2]; // Acceseaza al treilea element (val = 3)
  • Matrice

    • O matrice este un tablou bidimensional (sau multidimensional), in care datele sunt organizate pe linii si coloane. Este o extensie a tabloului unidimensional (sau vector).

    Definitie:

    int matrice[3][4]; // Matrice cu 3 linii si 4 coloane

    Initializare:

    int matrice[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

    In acest exemplu, matricea are 2 linii si 3 coloane, unde fiecare element poate fi accesat folosind doi indici, de exemplu: matrice[0][1] pentru accesarea valorii 2.

  • Struct în C++

    • În C++, structurile sunt o modalitate de a grupa variabile de tipuri diferite într-un singur obiect. Acestea sunt utile pentru a reprezenta entități complexe, având atribute de tipuri variate.

    Exemple de utilizare:

      Definire structură:

      struct Persoana {
    string nume;
    int varsta;
};

      Inițializare:

      Persoana persoana1 = {"Ion", 25};

      Accesare element:

      cout << persoana1.nume << endl;

    Structurile sunt ideale pentru a organiza date legate între ele, permițând acces rapid la fiecare atribut.

  • Coada în C++

    • O coadă (queue) este o structură de date care urmează principiul FIFO (First In, First Out), adică primul element introdus este primul care va fi eliminat. Cozile sunt utilizate frecvent pentru gestionarea resurselor într-un mod ordonat și eficient, cum ar fi procesarea de taskuri sau manipularea unor evenimente într-un sistem.

    Exemple de utilizare:

      Definiție

      >#include <queue>

std::queue<int> coada;
coada.push(10);
coada.push(20);

      Adăugare elemente (push) 

      coada.push(10);  // Adaugă 10 în coadă
coada.push(20);  // Adaugă 20 în coadă

      Eliminare elemente (pop): 

      coada.pop();  // Elimină primul element (10)

      Accesarea elementului din fața cozii (front): 

      int element = coada.front();  // Obține valoarea 20, care este primul element

    În exemplul de mai sus, am creat o coadă de întregi, am adăugat două elemente și apoi am eliminat primul element din coadă. Cozile sunt foarte utile în algoritmi de tip procesare pe secvență, cum ar fi în implementarea unui sistem de procesare a cererilor într-un server sau în manipularea evenimentelor într-un program.

  • Clasa în C++

    • În C++, o clasă este o structură de date care permite definirea de obiecte ce pot conține atât date (atribute), cât și funcții (metode) care operează asupra acestora. Clasa este un concept esențial în programarea orientată pe obiecte (OOP), permițând gruparea de comportamente și stări într-o singură entitate. Prin utilizarea claselor, putem crea tipuri de date personalizate, care modelează comportamente și procese din lumea reală.

    Exemple de utilizare:

      În C++, o clasă este o structură de date care permite definirea de obiecte...

      class Persoana {
public:
    std::string nume;
    int varsta;

    Persoana(std::string n, int v) : nume(n), varsta(v) {}

    void afiseazaInformatii() {
        std::cout << "Nume: " << nume << ", Varsta: " << varsta << std::endl;
    }
};

      În acest exemplu, am creat o clasă Persoana care are două atribute: nume și varsta. De asemenea, am definit un constructor care inițializează aceste atribute la crearea unui obiect al clasei și o metodă afiseazaInformatii() care afișează informațiile despre persoană.

      Crearea unui obiect și utilizarea metodei: 

      int main() {
    // Creăm un obiect al clasei Persoana
    Persoana persoana1("Ion", 30);

    // Apelăm metoda pentru a afișa informațiile
    persoana1.afiseazaInformatii();

    return 0;
}

      În funcția main(), am creat un obiect persoana1 de tip Persoana și i-am atribuit valorile "Ion" pentru nume și 30 pentru vârstă. Apoi, am apelat metoda afiseazaInformatii() pentru a afișa aceste informații pe ecran.

    În exemplul de mai sus, am creat o coadă de întregi, am adăugat două elemente și apoi am eliminat primul element din coadă. Cozile sunt foarte utile în algoritmi de tip procesare pe secvență, cum ar fi în implementarea unui sistem de procesare a cererilor într-un server sau în manipularea evenimentelor într-un program.

  • Enumerare în C++

    • În C++, o enumerare (sau enum) este o structură de date...

    enum ZiSaptamana {
    Luni, Marti, Miercuri, Joi, Vineri, Sambata, Duminica
};

int main() {
    ZiSaptamana zi = Joi;
    if (zi == Joi) {
        std::cout << "Este Joi!" << std::endl;
    }
    return 0;
}

    În acest exemplu, am definit o enumerare numită ZiSaptamana care conține zilele săptămânii. Fiecare zi este asociată cu o valoare întreagă, începând de la 0 (pentru Luni) până la 6 (pentru Duminica).

    Valorile implicite ale enumerării: Dacă nu sunt specificate valori pentru membrii enumerării, acestea vor începe de la 0 și vor crește cu 1 pentru fiecare membru succesiv.

    Modificarea valorilor membrilor: 

    enum Status {
    Inactiv = 0,   // Inactiv are valoarea 0
    Activ = 1,     // Activ are valoarea 1
    Suspendat = 5, // Suspendat are valoarea 5
    InAsteptare = 10 // InAsteptare are valoarea 10
};

    În acest exemplu, valorile membrilor sunt personalizate. De exemplu, Suspendat are valoarea 5 și InAsteptare are valoarea 10, în loc de valori implicite consecutive.

    Utilizarea enumerărilor în condiții: 

    
enum StareSemafor {
    Rosu,
    Galben,
    Verde
};

int main() {
    StareSemafor semafor = Verde; // Setăm semaforul pe Verde

    if (semafor == Verde) {
        std::cout << "Poți să treci!" << std::endl;
    } else if (semafor == Galben) {
        std::cout << "Atenție! Pregătește-te să oprești!" << std::endl;
    } else {
        std::cout << "Oprește!" << std::endl;
    }

    return 0;
}

    În acest exemplu, am creat o enumerare StareSemafor care definește cele trei stări ale unui semafor: Rosu, Galben și Verde. În funcție de valoarea setată a semaforului, afișăm un mesaj corespunzător.

    În exemplul de mai sus, am creat o coadă de întregi, am adăugat două elemente și apoi am eliminat primul element din coadă. Cozile sunt foarte utile în algoritmi de tip procesare pe secvență, cum ar fi în implementarea unui sistem de procesare a cererilor într-un server sau în manipularea evenimentelor într-un program.

  • String-uri în C++

    • În C++, un string este o secvență de caractere care poate fi manipulată și gestionată într-un mod mai ușor decât folosind un tablou de caractere (char[]). C++ oferă clasa std::string din biblioteca standard () pentru a lucra cu șiruri de caractere. Această clasă permite manipularea șirurilor de caractere în moduri foarte flexibile, fiind mult mai convenabilă decât lucrul cu tablouri de caractere tradiționale.

      1. Definiție și includere Pentru a folosi std::string, trebuie să incluzi biblioteca '': 

      
#include <iostream>
#include <string>

      2. Crearea unui string
      Există mai multe moduri de a crea un obiect de tip std::string:

      Inițializarea cu un șir de caractere literal:

      
std::string text = "Salut, lume!";

      Inițializarea unui string cu un anumit număr de caractere:

      
std::string text(5, 'A');  // Crează un string "AAAAA"

      3. Operații comune cu stringuri

      a. Afișarea unui string
      Poți folosi std::cout pentru a afișa un string: 

      
std::string text = "Salut, lume!";
std::cout << text << std::endl;

      b. Concatenarea stringurilor
      Poți concatena două sau mai multe stringuri folosind operatorul +: 

      
std::string salut = "Salut";
std::string nume = "Maria";
std::string mesaj = salut + ", " + nume + "!";
std::cout << mesaj << std::endl;  // Afișează "Salut, Maria!"

      c. Accesarea caracterelor din string
      Poți accesa caracterele individuale dintr-un string folosind indexul, la fel ca într-un tablou: 

      
std::string text = "Salut!";
char primulCaracter = text[0];  // 'S'
std::cout << primulCaracter << std::endl;

      d. Modificarea unui string: 

      
std::string text = "Salut!";
text[0] = 's';  // Schimbă 'S' cu 's'
std::cout << text << std::endl;  // Afișează "salut!"

      e. Obținerea lungimii unui string: 

      
std::string text = "Salut!";
std::cout << "Lungimea textului este: " << text.size() << std::endl;  // 6

  • Probleme Coada

    • Cerinta

    Sa se scrie un program care gestioneaza o coada de numere intregi. Initial coada este vida. Programul va citi de la tastatura o lista de operatii, care pot fi:

    push X – adauga valoarea intreaga X in coada;

    pop – elimina elementul din coada;

    front – afiseaza elementul de la inceputul cozii.

    Programul va realiza asupra cozii operatiile citite, in ordine. Afisarile se fac pe ecran, cate o valoare pe linie.

    Date de intrare

    Programul citeste de la tastatura un numar n, apoi cele n operatii.

    Date de iesire

    Programul va afisa pe ecran numerele rezultat ale operatiilor front, cate unul pe o linie.

    Restrictii si precizari

    Coada nu va contine la un moment dat mai mult de 1000 elemente.

    Elementele cozii vor fi cuprinse intre -1.000.000 si 1.000.000.

    Daca la un moment dat se aplica operatia pop si coada nu contine niciun element, operatia nu are efect.

    Daca la un moment dat se aplica operatia front si coada nu contine niciun element, operatia nu are efect (nu se afiseaza nimic).
